СИСТЕМНА СПЕЦИФІКАЦІЯ ТА ІНСТРУКЦІЯ ЕКОСИСТЕМИ
Про проєкт: Це повноцінне низькорівневе середовище розробки та емуляції архітектури CHIP-8. Серце системи — високопродуктивне інтерпретуюче ядро процесора та віртуальна файлова система (VFS), які написані на мові C і трансльовані за допомогою Emscripten у бінарний код WebAssembly (WASM). Це забезпечує нативну швидкість виконання інструкцій (60 FPS) та прямий доступ до пам'яті через JavaScript-буфер `Uint8ClampedArray`.
Моніторинг та Дисплей: Ліва панель містить оригінальний двоколірний екран (64x32 пікселів) із попіксельним рендерингом (pixel-accurate). Поруч із ним розташована жива Карта пам'яті (Memory Map), яка візуалізує всі 4096 байт RAM процесора. Ви можете наводити курсор для зчитування адреси (Addr) та поточних значень (Val) у реальному часі.
Набір інструментів розробника (IDE): Нижня панель об'єднує інтерактивний відладчик (Debug Panel) для контролю регістрів V0–VF, лічильника команд (PC) та таймерів, а також Стек даних (SPD Stack) для відстеження байтів у стеку.
Інтегрований Асемблер: Справа розташований текстовий редактор із підсвічуванням синтаксису. Ви можете писати код мовою мнемонік CHIP-8 (наприклад, `LD V0, 0xFF`, `CLS`), компілювати його прямо у браузері кнопкою Compile & Run та зберігати файли конфігурації у форматі `.asm`.
Віртуальний диск (FAT/VFS): Проєкт емулює енергонезалежне сховище розміром 1 МБ. Компільовані бінарні файли та збережені асемблерні коди записуються у віртуальну таблицю файлів, яка синхронізується з локальним сховищем браузера (IndexedDB). Ви можете запускати або видаляти збережені файли прямо зі списку.
Запуск ігор: Перетягніть будь-який готовий дамп гри (файл `.ch8`) у нижню область LOAD ROM або натисніть на неї для вибору файлу, щоб миттєво ініціалізувати емулятор.
